# 关于钟乐

这是一个温馨和谐的服务器, 是我们不断奋斗下创造出来的, 我们与玩家们一起度过了很多愉快的时光,在这里的日子将会是你值得回忆的美好。

我们不会做影响平衡、破坏公平的行为。欢迎玩家提出意见或者举报违规行为, 请大家自觉遵守游戏规则。

> “关山万里远征人, 一望关山泪满巾”
